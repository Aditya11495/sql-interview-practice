# SQL Interview Practice

This repository contains a structured collection of SQL queries for **interview preparation** and **real-world data analysis practice**.

It covers all important SQL concepts from **basic to advanced level**, with a focus on **corporate use cases**.

# Topics Covered

- Basic Queries (SELECT, WHERE, ORDER BY)
- Aggregate Functions (SUM, AVG, COUNT)
- GROUP BY & HAVING
- Joins:
  - INNER JOIN
  - LEFT JOIN
  - RIGHT JOIN
  - FULL JOIN
- Subqueries
- Window Functions:
  - RANK()
  - DENSE_RANK()
  - ROW_NUMBER()
- Advanced Problems:
  - Nth Highest Salary
  - Duplicate Records
  - Data Filtering & Cleaning
